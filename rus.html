Русский язык

Упражнение 446

Задание: сгруппировать относительные прилагательные.

а) По материалу:
деревянный, глиняный, серебряный, стеклянный, соломенный, кожаный.

б) По месту:
городской, сельский, подземный, передний, задний, верхний, нижний.

в) По времени:
сегодняшний, завтрашний, осенний, зимний, вечерний, утренний, давнишний.

г) По действию / назначению:
отрывной, обледенелый, точильный, набивной, нагревательный, временный, семенной.

Примеры:
языковое явление;
языковая колбаса.
